## Overview
Clash of Titans is a real-time 2D shooter game developed using C++ and the SFML library.  
The player controls a ground gun and shoots down incoming enemy planes while avoiding falling bombs.

## Features
- Event-driven game loop with real-time rendering
- Keyboard-controlled player movement and shooting
- Enemy planes with randomized bomb drops
- Collision detection between bullets, enemies, and player
- Score tracking and pause functionality

## Technologies Used
- C++
- SFML

## How to Run

```bash
g++ main.cpp -o game -lsfml-graphics -lsfml-window -lsfml-system
